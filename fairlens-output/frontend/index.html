What FairLens Detects

📐
Disparate Impact
EEOC 80% rule. Detects when selection rates differ significantly across protected groups.
⚖️
Demographic Parity
Measures selection rate gap between privileged and unprivileged groups in absolute terms.
🎯
Equal Opportunity
True positive rate gap across groups. Are qualified candidates equally likely to be selected?
🔍
Proxy Features
Detects features correlated with sensitive attributes that can re-introduce bias indirectly.
